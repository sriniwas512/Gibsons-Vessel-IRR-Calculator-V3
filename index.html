What's New

Vessel IRR Sensitivity Calculator — recent updates

Latest Release — April 2026

  • Fix Edge-case IRR now shows N/A. Unreliable results in extreme scenarios are suppressed — affected cells display N/A on the heatmap instead of a misleading number.
  • Fix Zero interest-rate crash fixed. Entering 0% interest rate previously produced NaN across the entire grid. Loan payments at 0% now calculate correctly as equal principal instalments.
  • New Input validation. The calculator now checks for invalid inputs (vessel age ≥ scrap age, flipped price/TC ranges, out-of-range equity %, blank fields, base price outside the grid) and shows a clear error before calculating.
  • UI Tokyo Night theme. The interface and heatmap canvas have been restyled with the Tokyo Night colour palette for a cleaner, high-contrast look.
  • Fix Earning days guard. Off-hire days during DD/SS can no longer push earning days below zero, which previously produced negative revenue and distorted IRR results.
